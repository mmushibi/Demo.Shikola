# Shikola Demo - School Management System

A simple school management system for schools with different user portals.

## About This App

Shikola Demo shows how a complete school management system works. It helps schools run better and makes talking to everyone easy.

## Main Features

### 🎓 **Different User Portals**
- **School Admin**: Manages students, teachers, classes, and fees
- **Teacher**: Takes attendance, gives grades, makes tests, and sends messages
- **Student**: Sees homework, results, timetable, and messages
- **Accountant**: Manages fees, expenses, and reports
- **Public Website**: Main page with information about the school

### 🛡️ **Security & Privacy**
- Blocks unauthorized access
- Stops copying and saving
- Checks if user is allowed to view pages
- Detects and blocks developer tools
- **Auto-deletes cookies, cache, and history** when browser closes
- **Cleans data every 5 minutes** for privacy

### 📱 **Easy to Use**
- Works on phones, tablets, and computers
- Clean and simple design
- Easy to find what you need
- Shows updates right away

### 🔧 **How It Works**
- Demo version - no internet needed
- All data is fake (for showing only)
- Changes disappear when you close the browser
- Pretend API calls for testing

## How to Start

1. Download the files
2. Open `index.html` in your browser
3. Click on different portals to explore
4. No login needed for demo

## What It's Made With

- **Frontend**: HTML, CSS, JavaScript
- **Style**: Tailwind CSS
- **Icons**: Font Awesome
- **Security**: Custom protection system

## Important Notes

This is just a demo to show how the system works. All data is fake and will disappear when you close the browser. The app automatically cleans your browser data to protect your privacy.
