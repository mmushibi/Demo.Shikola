Important Instructions
  • Your CSV data should be in the same column order as the downloaded template. The first line of your CSV must be the column headers as in the table example.
  • Make sure your file is saved as UTF-8 to avoid encoding problems.
  • If a column is a date, it must use the format Y-m-d (e.g. 2026-06-06).
  • Duplicate Registration Number rows will not be imported.
  • Please follow this date format 2026-06-15 for Date of birth & Admission date.
  • Only CSV files are allowed.
